Code With Consequence: Using Data for Good

Why I Write Code

I don’t write code for the sake of clever algorithms. I write code because it solves problems—because it has consequence.

Learning Through a Crisis

In 2020, I worked as a research intern at IIT Madras to understand how COVID-19 disrupted the energy economy for Indian MSMEs. We sifted through datasets, ran regressions, and visualized sectoral trends. But more than that, we gave policymakers tools to see where the real damage was.

Designing for the Underserved

Whether building a WaSH data tracker for GramVikas or designing a dashboard for NGO use in rural India, my focus is always impact first. If a solution only works for data scientists, it’s incomplete. Real-world tools must be rugged, intuitive, and usable by the people who need them most.

AI, Optimization, and the Physical World

My undergraduate thesis explored using artificial neural networks for aerodynamic shape optimization—essentially letting machines assist in how we design efficient forms. I believe machine learning belongs not just in apps, but in the physical spaces we move through daily.

What “Impact” Means to Me

It means designing code that changes how decisions are made. It means bringing intelligence to places where it’s never been. It means using data not just to predict, but to protect.

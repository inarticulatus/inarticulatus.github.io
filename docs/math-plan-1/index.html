📅 Day 1: Group Basics & Lagrange’s Theorem

Resources:

  • Artin: §2.1–2.3 | Herstein: §2.4–2.5

Goals:
✓ Define group, subgroup, coset
✓ Prove Lagrange’s Theorem: |G| = [G:H] · |H|

Key Examples:

  • ℤ/6ℤ: Subgroups = {0,3}, {0,2,4}, cosets
  • S₃: [S₃ : A₃] = 2

Exercise:
Find all subgroups of ℤ/9ℤ (Artin p.51 #1).

📅 Day 2: Normal Subgroups & Quotient Groups

Resources:

  • Artin: §2.8–2.9

Goals:
✓ Define normal subgroup (gHg⁻¹ = H)
✓ Construct quotient group G/N

Key Examples:

  • A₃ ◁ S₃ → S₃/A₃ ≅ ℤ/2ℤ
  • ℝ/ℤ ≅ circle group

Exercise:
Show SLₙ(ℝ) ◁ GLₙ(ℝ) (Artin p.79 #4).

📅 Day 3: Group Actions & Orbit-Stabilizer

Resources:

  • Artin: §6.1–6.3

Goals:
✓ Define group action (G ↷ X)
✓ Prove Orbit-Stabilizer: |G| = |Orb(x)|·|Stab(x)|

Key Examples:

  • S₃ acts on {1,2,3}: Orbits = {1,2,3}, Stab(1) = {e,(23)}
  • D₄ acts on square vertices

Exercise:
Classify orbits of ℤ ↷ ℝ² via n·(x,y) = (x+n, y) (Artin p.193 #1).

📅 Day 4: Sylow Theorems

Resources:

  • Artin: §6.7 | Herstein: §2.12

Goals:
✓ Sylow I: If pᵏ | |G|, ∃ subgroup of order pᵏ
✓ Sylow II: All Sylow p-subgroups are conjugate

Key Examples:

  • S₄: Sylow 2-subgroups (order 8)
  • A₅: No subgroup of order 15

Exercise:
Prove no group of order 36 is simple (Herstein p.94 #10).

📅 Day 5: Rings & Ideals

Resources:

  • Artin: §11.1–11.3

Goals:
✓ Define ring, ideal, quotient ring
✓ Prove R/I is a ring

Key Examples:

  • ℤ/nℤ: (n) is maximal ⇔ n prime
  • ℝ[x]/(x²+1) ≅ ℂ

Exercise:
Find all ideals of ℤ × ℤ (Artin p.389 #2).

📅 Day 6: Field Extensions

Resources:

  • Dummit & Foote: §13.1–13.2

Goals:
✓ Define algebraic extension, minimal polynomial
✓ Prove ℚ(√2) is degree-2 extension

Key Examples:

  • ℚ(∛2): x³ - 2 irreducible
  • 𝔽₄ = 𝔽₂[x]/(x²+x+1)

Exercise:
Find [ℚ(√2+√3):ℚ] (Dummit p.519 #1).

📅 Day 7: Review & Anki Setup

Tasks:

  1. Create cheat sheet of theorems.
  2. Make Anki cards for:
    • Lagrange’s Theorem
    • Sylow III (# Sylow p-subgroups ≡ 1 mod p)
    • I maximal ⇔ R/I is a field
